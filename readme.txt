== Description ==

(OFFICIAL beecloud.id plugin) Membantu pencatatan order woocommerce anda, update harga, hingga update stok barang di woocommerce anda.

Lebih dari 1100+ pengusaha menggunakan Beecloud untuk mengontrol bisnisnya dimana  saja dan kapan saja.

Tidak perlu lagi repot riwa riwi hanya untuk kontrol cabang. Anda dapat memonitor cabang manapun dari genggaman handphone anda secara real time.

Mengapa menggunakan Beecloud ?
- Aman dari manupulasi dan kebocoran
- Dengan Beecloud anda tidak perlu belajar akunting selama 4 tahun untuk bisa buat laporan keuangan. Cukup 1x klik anda sudah bisa mendapatkan laporan keuangan secara lengkap.
- Bisa di akses dari HP, Tablet & PC
